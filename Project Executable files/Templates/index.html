ABOUT PROJECT

This Machine Learning Model is responsible for calculating credit card approval based on data like type of education, property, job profile, gender, credit score and repayment behaviour, etc. It is taking these data and based on previous customers' behaviour this model is analyzing if the applicant is eligible for taking a credit card or not.
